# Lab19_Web Hosting

* ### useradd / adduser / usermod / deluser
  * **user**: 시스템에 로그인한 사용자를 출력하는 명령어.
  * **adduser**: adduser[계정명]으로 입력. ==> 계정 + 비밀번호 + 홈디렉토리 생성(자세하게 생성가능)
  * **usermod**: usermod[옵션][계정명]으로 입력. ==>계정 정보 변경
  * **deluser**: deluser[계정명]으로 입력. ==> 계정 삭제
